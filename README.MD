# Predicting Chromophore Optical Properties Across Solvents Using Random Forest Regression

This repository contains the code for my project for the Digital Alchemy course.
The repository contains a single Jupyter notebook file which contains all the requirements, code, and results for the machine learning model

To run the notebook:

- Install conda
- Create a new conda environment
- Import dependencies using:

```
conda env create -f environment.yml
```

- run the notebook
